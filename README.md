# MomoSEO & Business Suite — GitHub Pages

Version compatible GitHub Pages.

## Fonctionnalités

- Dashboard sombre Tailwind CSS
- Demande de clé API Mistral au démarrage
- Générateur de balises SEO
- Assistant de réponses aux avis Google Business
- Historique local avec localStorage
- Boutons copier

## Important

Cette version ne nécessite pas PHP ni MySQL.

Elle fonctionne directement avec :

- `index.html`
- GitHub Pages
- JavaScript
- API Mistral

## Installation GitHub Pages

1. Envoyer `index.html` dans le dépôt GitHub.
2. Aller dans `Settings`.
3. Aller dans `Pages`.
4. Source : `Deploy from a branch`.
5. Branch : `main`.
6. Folder : `/root`.
7. Sauvegarder.

Ensuite ouvrir l'URL GitHub Pages.
